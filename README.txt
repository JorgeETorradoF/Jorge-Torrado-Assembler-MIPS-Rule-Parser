Descripción:

El presente proyecto es un rule parser en assembler MIPS, que lee 2 archivos, 1 con data de logs y el otro con condiciones, compara cada log con cada una de las condiciones y si coincide lo manda a un archivo llamado Alerts_Triggered.log, adicionalmente se puede consultar un log por IP o username, generando un archivo con el log correspondiente si se encontró, de otra manera el archivo querdará vacío.




FAVOR LEER ESTA LINEA ANTES DE OPERAR EL CODIGO:

Antes de operar el rule parser favor cambie las rutas de acceso, configurándolas de acuerdo a la ruta donde quedaron ubicados los archivos al descargarlos a su pc
